# ESP32-S3 Dev Board — KiCad Project Files

Custom 4-layer ESP32-S3 development board designed in KiCad 9.
Manufactured by JLCPCB using the JLC04161H-3313 stackup.

## Contents
- /Schematic — KiCad schematic files
- /PCB — KiCad PCB layout files  
- /Gerbers — Manufacturing files for JLCPCB
- /BOM — Bill of materials
